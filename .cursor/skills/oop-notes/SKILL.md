# OOP 筆記主軸

## 何時使用
- 新增/修改 Java OOP 相關頁面或說明（`OopPage` 或對應內容）
- 撰寫類別設計、繼承、多型、封裝、介面等教學與範例

## 專案脈絡
- 本專案為前端筆記站，OOP 內容以**說明頁／教學文件**形式呈現。
- 程式碼範例以 Java 為主，簡潔、可讀，並配合簡短註解。

## 撰寫要點
- **結構**：章節分明（如 OOP 概念、四大特性、類別與物件、繼承與多型、介面與抽象類別、範例程式碼）。
- **術語**：繁體中文為主，術語可中英並列（如「封裝（Encapsulation）」）。
- **範例**：類別命名 PascalCase、方法/變數 camelCase、常數 ALL_CAPS；可配合 UML 或流程說明。
- **格式**：與專案其他頁面一致—深色主題、程式碼區塊使用既有樣式。

## 與其他主軸配合
- 版面與元件風格遵循 [frontend-ui](../frontend-ui/SKILL.md)；章節與程式碼區塊可參考 [algorithm-lab](../algorithm-lab/SKILL.md)。
